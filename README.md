# Personal Website Redesign

This folder contains a static personal website redesign for `yingjundong.com`.

## Files

- `index.html` - homepage content and structure
- `styles.css` - gray visual theme and responsive layout
- `script.js` - mobile navigation and email-copy behavior

## Upload

Upload these files to the root of your website hosting location:

```text
index.html
styles.css
script.js
```

If your host already has an `index.html`, replace it with this version after backing up the current file.

## CV Link

The `Download CV` button points to:

```text
Yingjun_Dong_Professional_Resume_2026.docx
```

Upload that file to the same website folder, or change the link in `index.html` to the correct CV file path.

## Notes

- The page is build-free: no npm, React, or bundler required.
- The design uses Google Fonts. If you prefer fully local assets, remove the Google Fonts lines in `index.html`; the site will fall back to system fonts.
- Replace the featured publication cards with any updated publications you want to highlight.
